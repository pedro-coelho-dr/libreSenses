Bem-vindo ao LibreSenses

  • Esta página é o ponto de partida do protótipo de nossa solução: uma ferramenta de submissão de conteúdo acessível para o LibreFlix.
  • O objetivo desta página é facilitar a navegação e a compreensão das funcionalidades da ferramenta.
  • Aqui, os usuários podem submeter um novo filme utilizando nosso formulário ou acessar filmes já cadastrados para adição de elementos que tornem o conteúdo mais acessível.
  • Obs.: Para fins de facilitar testes, alguns campos do formulário não estão como obrigatórios quando deveriam estar.
  • Obs 2.: Para facilitar teste, os filmes cadastrados ficam disponíveis para todos os usuários.
  • Obs 3.: No formulário de envio de filmes, propor algumas mudanças.
